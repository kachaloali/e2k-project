# Theme Wordpress

Ceci est le thème Wordpress utilisé sur la majorité des sites de JLM 2017. Il est basé sur [Sage](https://roots.io/sage/). Merci de lire la documentation de Sage pour comprendre comment construire et utiliser ce theme.

Ce thème dépend du [plugin Wordpress JLM 2017](https://github.com/jlm2017/wp-plugin). Ce plugin doit être installé pour que le thème fonctionne correctement.
